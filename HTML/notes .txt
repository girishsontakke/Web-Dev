CSS cheat sheet

Cascading Style Sheet at the most basic level indicates that the order of css rules matter.


.class
#id 
*  <!-- for all  elements -->
element
element, element2  (for both the elements)
element element2  (for element2 inside element)
element + element2 (for element2 next to element)
element > element2 (for element2 having parent as element)
:hover  (changes when we move curser to element)
:last - child ( for same class element)
:first-child
!important
